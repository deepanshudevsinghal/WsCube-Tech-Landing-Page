# 🎓 WsCube Tech Landing Page Clone

A fully responsive **WsCube Tech Landing Page Clone** built using **HTML5, CSS3, and Bootstrap 5**.  
This project replicates a modern educational website landing page and demonstrates strong skills in **Figma to HTML conversion**, responsive design, and UI implementation.

## 🚀 Live Demo
 https://deepanshudevsinghal-debug.github.io/WsCube-Tech-Landing-Page/

## 📌 Project Overview

This project is a frontend clone of the WsCube Tech educational platform landing page.  
The layout was designed in Figma and converted into a fully responsive webpage using Bootstrap and custom CSS.

---

## ✨ Features

- ✅ Fully responsive landing page
- ✅ Figma to HTML conversion
- ✅ Bootstrap 5 components
- ✅ Navigation bar with CTA buttons
- ✅ Hero section with structured layout
- ✅ Course showcase section
- ✅ Testimonials / Features section
- ✅ Footer with structured links
- ✅ Clean and modern UI design
